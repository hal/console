Jakarta Bean Validation은 Java 사양으로,

  • 주석을 통해 객체 모델에 대한 제약 조건을 표현합니다.
  • 확장 가능한 방식으로 맞춤 제약 조건을 작성합니다.
  • 객체 및 객체 그래프의 유효성을 검사하는 API를 제공합니다.
  • 메소드와 생성자의 매개변수와 반환 값을 검증하는 API를 제공합니다.
  • 위반 사항 보고(현지화) 합니다.
  • Java SE에서 실행되며 Jakarta EE 8에 통합됩니다.

${build.fullName}은 구현을 위해 Hibernate 유효성 검사기를 사용합니다.
