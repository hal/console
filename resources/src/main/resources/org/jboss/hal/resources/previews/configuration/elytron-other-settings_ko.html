이 섹션은 아래에 설명된 여러 리소스로 구성됩니다:

인증 구성

${build.fullName}에 배포된 클라이언트와 원격 연결 시 인증을 위한 기타 리소스에 사용되는 개별 인증 구성 정의입니다.

인증 컨텍스트

${build.shortName}에 배포된 클라이언트와 기타 리소스가 원격 연결을 설정할 때 SSL 컨텍스트 및 인증 구성을 제공하는 데 사용되는 개별 인증 컨텍스트 정의입니다.

저장소

키 저장소: 키 저장소 유형, 위치, 액세스 자격 증명을 포함하는 키 저장소 또는 신뢰 저장소에 대한 정의입니다. 외부 서비스의 비밀번호와 같은 민감한 정보에 대한 별칭을 유지하기 위한 자격 증명 저장소입니다.

SSL

SSL 연결의 클라이언트측과 서버측을 구성합니다. SSL 컨텍스트를 생성하는 데 사용되는 키 관리자 목록을 생성하기 위한 키 관리자 정의입니다. 사용 가능한 기타 리소스: 보안 속성, 프로바이더 로더, 애그리거트(Aggregate) 프로바이더.

보안 도메인

보안 도메인은 하나 이상의 보안 영역과 변환을 수행하는 리소스 집합으로 뒷받침되는 보안 정책을 나타냅니다.

로그

파일 및 syslog에 지속되는 감사 로거.
