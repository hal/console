IO 서브시스템은 Undertow 및 Remoting과 같은 다른 서브시스템에서 사용되는 XNIO 워커 및 버퍼 풀을 정의합니다.

워커

워커는 XNIO 워커 인스턴스입니다. XNIO 워커 인스턴스는 IO 및 워커 스레드 관리, SSL 지원과 같은 기능을 제공하는 Java NIO API의 추상화 계층입니다. 기본적으로 ${build.shortName}은 다음과 같은 단일 워커를 제공합니다. 기본값이지만 더 많이 정의할 수 있습니다.

버퍼 풀

버퍼 풀은 풀링된 NIO 버퍼 인스턴스입니다.

중요:
버퍼 크기를 변경하면 애플리케이션 성능에 큰 영향을 미칩니다. 대부분의 서버에서 이상적인 크기는 일반적으로 16k입니다.
