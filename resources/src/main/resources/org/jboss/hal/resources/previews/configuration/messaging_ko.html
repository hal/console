Apache ActiveMQ Artemis

Apache ActiveMQ Artemis는 비동기 메시징 시스템을 위한 오픈 소스 프로젝트입니다. 고성능이고 내장 가능하며 클러스터링되어 있으며 여러 프로토콜을 지원합니다. ${build.shortName}은 Apache ActiveMQ Artemis를 JMS 브로커로 사용하며 message-activemq 서브시스템을 사용하여 구성됩니다. 이는 HornetQ 브로커를 완전히 대체하지만 이전 버전과의 프로토콜 호환성을 유지합니다.

코어 ActiveMQ Artemis는 JMS에 구애받지 않으며 코어 API라고 하는 비JMS API를 제공합니다. ActiveMQ Artemis는 또한 파사드 계층을 사용하여 코어 API 위에 JMS 의미 체계를 구현하는 JMS 클라이언트 API를 제공합니다. 기본적으로 JMS 상호 작용은 JMS 클라이언트 API를 사용하여 클라이언트 측의 코어 API 작업으로 변환됩니다. 여기에서 모든 작업은 코어 클라이언트 API 및 Apache ActiveMQ Artemis 와이어 형식을 사용하여 전송됩니다. 서버 자체는 코어 API만 사용합니다. 코어 API와 해당 개념에 대한 자세한 내용은 다음을 참조하세요.
