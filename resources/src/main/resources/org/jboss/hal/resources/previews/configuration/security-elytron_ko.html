elytron 서브시스템은 ${build.fullName}에 새로 추가되었습니다. 이는 전체 애플리케이션 서버에서 보안을 통합하는 데 사용되는 보안 프레임워크인 WildFly Elytron 프로젝트를 기반으로 합니다. elytron 서브시스템을 사용하면 단일 구성 지점에서 애플리케이션과 관리 인터페이스를 모두 보호할 수 있습니다. WildFly Elytron은 맞춤형 기능 구현을 제공하고 서브시스템과 통합하기 위한 API 및 SPI 세트도 제공합니다.

또한 Elytron에는 다음과 같은 몇 가지 중요한 기능이 있습니다.

  • HTTP 및 SASL 인증을 위한 더욱 강력한 인증 메커니즘.
  • SecurityIdentities가 보안 도메인 전체에 전파될 수 있도록 아키텍처가 개선되었습니다. 이렇게 하면 인증에 사용할 수 있는 투명한 변환이 보장됩니다. 이러한 변환은 구성 가능한 역할 디코더, 역할 매퍼 및 권한 매퍼를 사용하여 이루어집니다.
  • 암호화 제품군 및 프로토콜을 포함한 SSL/TLS 구성을 위한 중앙 집중식 지점입니다.
  • SSL/TLS 연결과 인증을 밀접하게 연결하는 Eager SecureIdentity와 같은 SSL/TLS 최적화를 설정합니다 . Eager SecureIdentity 구성을 사용하면 요청별로 SecureIdentity를 구성할 필요가 없습니다. 인증을 SSL/TLS 연결 설정과 밀접하게 연결하면 첫 번째 요청이 수신되기 전에 권한 확인이 수행될 수 있습니다.
  • 이전 Vault 구현을 대체하는 보안 자격증명 저장소입니다. 새로운 보안 자격증명 저장소는 암호화된 문자열 외에도 여러 가지 암호화된 자격증명 유형을 저장할 수 있습니다.

새로운 elytron 서브시스템은 레거시 보안 서브시스템 및 레거시 코어 관리 인증과 병행하여 존재합니다. 관리 인터페이스를 보호하고 애플리케이션에 대한 보안을 제공하기 위해 레거시 방법과 Elytron 방법을 모두 사용할 수 있습니다.
