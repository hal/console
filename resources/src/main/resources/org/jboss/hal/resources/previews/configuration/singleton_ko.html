싱글톤 배포 동작을 구성하거나 싱글톤 MSC 서비스를 생성하기 위한 싱글톤 정책을 정의합니다.

고가용성(HA) 싱글톤이라고도 알려진 클러스터형 싱글톤 서비스는 클러스터의 여러 노드에 배포되는 서비스입니다. 서비스는 노드 중 하나에서만 제공됩니다. 싱글톤 서비스를 실행하는 노드를 일반적으로 기본 노드라고 합니다.

기본 노드가 실패하거나 종료되면 나머지 노드에서 다른 기본 노드가 선택되고 새 기본 노드에서 서비스가 다시 시작됩니다. 하나의 기본이 중지되고 다른 기본이 아직 인계받지 않는 짧은 간격을 제외하고 서비스는 단 하나의 노드에 의해 제공됩니다.
