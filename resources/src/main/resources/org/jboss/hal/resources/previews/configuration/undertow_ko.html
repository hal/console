Undertow 서브시스템을 사용하면 웹 서버와 서블릿 컨테이너 설정을 구성할 수 있습니다. 이는 다음을 구현합니다. Java Servlet 3.1 사양 및 웹소켓을 지원하며 HTTP 업그레이드 및 서블릿 배포에서 고성능 비차단 핸들러를 사용합니다. Undertow 서브시스템에는 mod_cluster를 지원하는 고성능 역방향 프록시 역할도 할 수 있습니다.

Undertow 서브시스템 내에는 구성할 주요 구성 요소가 6개 있습니다.

  1. 전역 설정
  2. 버퍼 캐시
  3. 서버
  4. 서블릿 컨테이너
  5. 필터
  6. 핸들러

중요
Undertow 서브시스템은 IO 서브시스템을 사용하여 XNIO 작업자 및 버퍼 풀도 제공합니다. IO 서브시스템은 별도로 구성되며 대부분의 경우 최적의 성능을 제공하는 기본 구성을 제공합니다.
