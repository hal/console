JBossWS 구성 요소는 웹 서비스 서브시스템을 통해 애플리케이션 서버에 제공됩니다. JBossWS 구성 요소는 WS 엔트포인트 프로세스를 다룹니다. 서브시스템은 게시된 엔드포인트 주소 및 엔드포인트 처리기 체인의 구성을 지원합니다. 기본 웹 서비스 서브시스템은 서버의 도메인 및 독립형 구성 파일에 제공됩니다.
