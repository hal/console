콘텐츠 저장소는 도메인 컨트롤러에 업로드된 모든 콘텐츠를 보관합니다. 업로드된 콘텐츠는 서버 그룹에 할당될 수 있습니다.

드래그 앤 드롭을 사용하여 새 콘텐츠를 추가하거나 기존 콘텐츠를 교체할 수 있습니다. 하나 또는 여러 개의 파일을 콘텐츠 열로 드래그하기만 하면 됩니다. 동일한 이름의 콘텐츠가 이미 있는 경우 콘텐츠가 교체되고, 그렇지 않은 경우 콘텐츠가 추가됩니다.

관리되지 않는 배포를 추가할 수도 있습니다. 비관리형 배포는 서버의 로컬 파일 시스템에 있는 폴더를 가리킵니다. 관리형 배포와 비교하여 비관리형 배포는 배포되기 전에 서버의 배포 저장소에 복사(즉, 업로드)되지 않습니다. 배포 콘텐츠는 그대로 유지되며 원래 위치에서 직접 배포됩니다.

콘텐츠가 더 이상 서버 그룹에 할당되지 않은 경우 해당 콘텐츠를 다시 삭제할 수 있습니다.
