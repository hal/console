서버 그룹 배포 관리

다음을 사용할 수 있습니다. 선택한 서버 그룹에 새 배포를 추가하려면 드래그 앤 드롭하세요. 하나 또는 여러 개의 파일을 배포 열로 드래그하기만 하면 됩니다. 드래그 앤 드롭으로 추가된 배포는 기본적으로 비활성화됩니다.

콘텐츠 저장소에서 하나 이상의 항목을 선택하고 선택한 서버 그룹에 배포합니다.

관리되지 않는 배포를 추가합니다. 비관리형 배포는 서버의 로컬 파일 시스템에 있는 폴더를 가리킵니다. 관리형 배포와 비교하여 비관리형 배포는 배포되기 전에 서버의 배포 저장소에 복사(즉, 업로드)되지 않습니다. 배포 콘텐츠는 그대로 유지되며 원래 위치에서 직접 배포됩니다.
