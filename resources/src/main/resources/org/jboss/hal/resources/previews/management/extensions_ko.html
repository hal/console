확장 프로그램은 관리 콘솔에 새로운 기능을 추가하는 방법입니다. JavaScript로 작성되었으며 다음을 사용해야 합니다. JavaScript API를 사용하여 콘솔 및 관리 인터페이스와 상호작용합니다. 확장 기능을 개발하려면 다음을 참조하세요. 자세한 내용은 https://github.com/hal/hal.next/wiki/Extensions를 참조하세요.

확장 프로그램은 JavaScript로 작성되며 다음과 같습니다. 브라우저에 삽입됩니다. 신뢰할 수 있는 확장 프로그램만 설치하세요!

확장 포인트

콘솔은 확장 프로그램에서 사용할 수 있는 4가지 확장 지점을 제공합니다.

  1. 헤더: 헤더의 '확장 프로그램' 드롭다운에 메뉴 항목을 추가합니다.
  2. 찾기 항목: 특정 찾기 열에 새 항목을 추가합니다.
  3. 바닥글: 바닥글의 '확장 프로그램' 드롭다운에 메뉴 항목을 추가합니다.
  4. 사용자 정의: 콘솔에 자신을 추가하는 방법은 확장 프로그램에 달려 있습니다.

설치

다음 두 가지 방법을 사용하여 확장 프로그램을 콘솔에 추가할 수 있습니다.

번들 확장 프로그램

번들 확장은 ${build.fullName} 설치의 일부이며 모듈로 설치됩니다. 콘솔 외부에 설치해야 합니다. 번들 확장 프로그램을 추가하거나 제거한 후 WildFly와 콘솔을 다시 시작하거나 다시 로드해야 합니다.

독립형 확장

독립형 확장 프로그램은 공개적으로 사용 가능한 엔드포인트에서 호스팅됩니다. 이 엔드포인트는 확장에 대한 메타데이터가 포함된 JSON 파일을 제공해야 합니다. 관리 콘솔을 사용하여 독립 실행형 확장을 추가하고 제거할 수 있습니다. 브라우저의 로컬 저장소에 저장됩니다. 따라서 관리 콘솔을 실행하는 브라우저와 URL로 범위가 지정됩니다.
