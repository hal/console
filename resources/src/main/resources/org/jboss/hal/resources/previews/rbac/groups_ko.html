mgmt-users.properties 파일이나 LDAP 서버를 사용하여 인증된 사용자는 사용자 그룹의 구성원이 될 수 있습니다. 사용자 그룹은 한 명 이상의 사용자에게 할당할 수 있는 임의의 라벨입니다.

RBAC 시스템은 사용자가 어떤 사용자 그룹에 속해 있는지에 따라 사용자에게 자동으로 역할을 할당하도록 구성할 수 있습니다. 또한 그룹 멤버십에 따라 역할에서 사용자를 제외할 수도 있습니다.

mgmt-users.properties 파일을 사용하는 경우 그룹 정보는 mgmt-groups.properties 파일에 저장됩니다. LDAP를 사용할 때 그룹 정보는 LDAP 서버에 저장되며 LDAP 서버 담당자가 관리합니다.
