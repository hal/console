elytron 서브시스템의 다음 리소스에 대한 런타임 작업 세트를 제공합니다.

  • Credential Store: 별칭을 추가하고 읽고 제거하고 별칭에 대한 비밀번호를 설정합니다. 비밀 키를 생성하고 가져오고 내보냅니다. 또한 자격증명 저장소를 다시 로드합니다.
  • Filtering Key Store: 별칭을 읽고 제거합니다.
  • Key Store: 별칭을 읽고 변경하고 제거합니다. 인증서를 가져오고 내보냅니다. 키 쌍을 생성합니다. 인증서 서명 요청을 생성합니다. 키 저장소를 로드하고 저장합니다.
  • LDAP Key Store: 별칭을 읽고 제거합니다.
  • Secret Key Credential Store: 별칭을 읽고 제거합니다. 비밀 키를 생성하고 가져오고 내보냅니다.
