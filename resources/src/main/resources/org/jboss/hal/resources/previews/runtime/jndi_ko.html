로컬 JNDI 네임스페이스의 개요를 제공합니다. Jakarta EE 플랫폼 사양은 다음 JNDI 컨텍스트를 정의합니다.

  • java:comp - 네임스페이스 범위는 현재 구성 요소(예: EJB)로 지정됩니다.
  • java:module - 현재 모듈로 범위 지정
  • java:app - 현재 애플리케이션으로 범위 지정
  • java:global - 애플리케이션 서버로 범위가 지정됨

표준 네임스페이스 외에도 ${build.shortName}은 다음 두 가지 전역 네임스페이스도 제공합니다.

  • java:jboss
  • java:/

java:jboss/exported 컨텍스트 내의 항목만 원격 JNDI를 통해 액세스할 수 있다는 점에 유의하십시오. 웹 배포의 경우 java:comp는 java:module로 별칭이 지정되므로 war에 배포된 EJB에는 자체 comp 네임스페이스가 없습니다.
