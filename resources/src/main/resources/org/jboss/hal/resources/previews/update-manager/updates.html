JBoss EAP has periodic releases that contain bug and security fixes. You can update your existing installation of JBoss EAP in the following ways:

  • Online update - This option enables you to connect and get updates directly from an online Maven repository. You must have access to the internet to use the online option.
  • Offline using archives - This option enables you to perform the installation updates from a local or file-system Maven repository.
  • Custom patches - This option enables you to install the specially customized patches.
  • Custom - This options enables you to specify additional parameters.

You can also revert back the installation to a previous state.
